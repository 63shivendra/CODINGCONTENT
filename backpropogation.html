Backpropagation Derivation

Notation & Methodology based on GO Classes (Sachin Mittal) ML Lectures.

3-Layer MLP Topology

1. Notation Reference

Variables

  • \( net_{j} \): Pre-activation sum (\(\sum w_i x_i + b\))
  • \( out_{j} \): Post-activation output (\(\sigma(net_j)\))
  • \( E_{total} \): Total Error / Loss function

Gradients

  • \( \delta_j \): Error signal (\(\frac{\partial E}{\partial net_j}\))
  • \( \eta \): Learning rate
  • \( \Delta w \): Change in weight (\(-\eta \frac{\partial E}{\partial w}\))

2. Forward Pass (Phase 1)

Calculate activations for hidden neuron \( h_1 \):

$$net_{h1} = w_{h1x1}x_1 + w_{h1x2}x_2 + b_{h1}$$ $$out_{h1} = \frac{1}{1 + e^{-net_{h1}}}$$

Total Error for the network:

$$E_{total} = \sum \frac{1}{2}(target - out_{o})^2$$

3. Backprop Derivation (Phase 2)

Case 1: Weights between Hidden & Output (\( w_{oh} \))

We want to find how much \( w_{o1h1} \) affects \( E_{total} \):

$$\frac{\partial E_{total}}{\partial w_{o1h1}} = \underbrace{\frac{\partial E_{total}}{\partial out_{o1}}}_{\text{1}} \times \underbrace{\frac{\partial out_{o1}}{\partial net_{o1}}}_{\text{2}} \times \underbrace{\frac{\partial net_{o1}}{\partial w_{o1h1}}}_{\text{3}}$$
  • \(\frac{\partial E}{\partial out_o} = -(target - out_o)\)
  • \(\frac{\partial out}{\partial net} = out_o(1 - out_o)\) (for Sigmoid)
  • \(\frac{\partial net}{\partial w} = out_{h1}\)

Case 2: Weights between Input & Hidden (\( w_{hx} \))

This is the "Hidden Layer" update. We must sum contributions from all output errors:

$$\frac{\partial E_{total}}{\partial w_{h1x1}} = \left( \sum \frac{\partial E_{total}}{\partial net_{out}} \cdot \frac{\partial net_{out}}{\partial out_{h1}} \right) \cdot \frac{\partial out_{h1}}{\partial net_{h1}} \cdot \frac{\partial net_{h1}}{\partial w_{h1x1}}$$

4. The Update Rule

Once gradients are calculated, we shift the weight in the direction of steepest descent:

$$w_{new} = w_{old} - \eta \cdot \frac{\partial E_{total}}{\partial w}$$

Key Intuition from Sachin Mittal's Lectures:

Always focus on the Local Gradient (\(\delta\)). The derivative of a node is always its incoming activation multiplied by the error signal traveling back from the next layer. This "caching" of \(\delta\) values is what makes backpropagation efficient.
