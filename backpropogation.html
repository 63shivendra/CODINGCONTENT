Backpropagation Neural Network – Complete Diagram & Derivation

Forward Propagation

net₃ = x₁w₁₃ + x₂w₂₃ net₄ = x₁w₁₄ + x₂w₂₄ y₃ = σ(net₃) y₄ = σ(net₄) net₅ = y₃w₃₅ + y₄w₄₅ y₅ = σ(net₅)

Loss Function

E = ½ (t − y₅)²

Backpropagation

δ₅ = (y₅ − t) · y₅(1 − y₅) δ₃ = δ₅ · w₃₅ · y₃(1 − y₃) δ₄ = δ₅ · w₄₅ · y₄(1 − y₄)

Weight Updates

w₃₅ ← w₃₅ − η·δ₅·y₃ w₄₅ ← w₄₅ − η·δ₅·y₄ w₁₃ ← w₁₃ − η·δ₃·x₁ w₂₃ ← w₂₃ − η·δ₃·x₂ w₁₄ ← w₁₄ − η·δ₄·x₁ w₂₄ ← w₂₄ − η·δ₄·x₂
